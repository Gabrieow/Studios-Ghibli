Esse é um projeto inspirado no site oficial da [Studio Ghibli Brasil](https://studioghibli.com.br/studioghibli/), porém, com novas funcionalidades
e a integração da API da Studio Ghibli, contendo informações sobre filmes, personagens e locais emblemáticos da produtora.
O Site conta com 4 temas de estilo diferentes, menu off-screen, animações AOS e CSS e sistema de criação de conta para favoritar seus filmes favoritos.

O site ainda está em desenvolvimento e está sendo utilizado primeiramente como forma prática de aprender novos conceitos no Next.JS.

## Inicializando projeto localmente

Primeiro, insira a seguinte linha de comando no console do projeto:

```
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
